# 🌸 Интернет-магазин «Bloom» 

Интернет-магазин для продажи цветов, клубники в шоколаде и сладких букетов. Проект разработан в учебных целях для отработки навыков фронтенд-разработки и демонстрации подхода к созданию интерактивных веб-приложений.

## ✨ О проекте

Многостраничный сайт с динамической корзиной, который позволяет пользователям просматривать каталог товаров, добавлять их в корзину и оформлять заказ. Проект реализован на чистом JavaScript с фокусом на чистоту кода, модульность и удобство пользователя.

**Цель проекта:** Продемонстрировать навыки верстки, написания логики на JavaScript, управления состоянием приложения (на примере корзины) и создания адаптивных интерфейсов.

## 🛠 Технологии и инструменты

*   **HTML5:** Семантическая верстка страниц.
*   **CSS3:**
    *   Flexbox для построения сеток и позиционирования элементов.
    *   Медиа-запросы (Media Queries) для обеспечения адаптивности и корректного отображения на всех устройствах (mobile-first подход).
    *   Модульная структура CSS-файлов (общие стили вынесены в `header_footer.css`, стили для конкретных страниц — в отдельные файлы).
*   **JavaScript (ES6+):**
    *   Работа с DOM-деревом (поиск, создание, изменение элементов).
    *   Обработка событий (клики по кнопкам, отправка форм).
    *   Работа с массивами объектов и методы высшего порядка (`forEach`, `filter`, `reduce`).
    *   Регулярные выражения для обработки текста (парсинг цены).
    *   Web Storage API (`localStorage`) для сохранения состояния корзины между сессиями.
*   **Git:** Система контроля версий для отслеживания изменений и организации процесса разработки.


## ⚙️ Функциональность и особенности реализации

### 📱 Адаптивный дизайн
*   Использован подход **mobile-first**. Все страницы корректно перестраиваются для экранов шириной до 768px и более.
*   Пример: на странице "О нас" блок с изображением и текстом на мобильных устройствах выстраивается вертикально (`flex-direction: column`), обеспечивая удобное чтение.

### 📝 Форма заказа
*   На странице `payDelivery.html` реализована форма для ввода адреса доставки.
*   Применена базовая валидация с использованием HTML5-атрибутов (`required`, `pattern` для поля с индексом).
